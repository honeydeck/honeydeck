# Use Beams as slide background

Honeydeck scales slides with CSS transforms.

Canvas/WebGL backgrounds measure their container, so use inverse scaling.

Result: the effect sees the logical slide size, not the transformed display size.

</Left>
<Right>

```tsx
const { slideWidth, slideHeight } = useHoneydeck();
const slideScale = useSlideScale();

<div
  className="absolute left-0 top-0"
  style={{
    width: slideWidth / slideScale,
    height: slideHeight / slideScale,
  }}
>
  <Beams />
</div>
```

</Right>

Find more backgrounds at [reactbits.dev](https://reactbits.dev/get-started/introduction).
