# LaTeX in user space

Honeydeck can render LaTeX through a normal project dependency and a tiny deck-local React component.

<div className="grid gap-5 text-[0.9em]">
  <p>
    Inline math stays in prose: <InlineLatex source={String.raw`E = mc^2`} /> and <InlineLatex source={String.raw`a^2 + b^2 = c^2`} />.
  </p>

  <BlockLatex source={String.raw`\int_{-\infty}^{\infty} e^{-x^2}\,dx = \sqrt{\pi}`} />
</div>

# Display equations

Use a block component when the equation is the point of the slide.

<BlockLatex source={String.raw`
\begin{aligned}
\nabla \cdot \vec{E} &= \frac{\rho}{\varepsilon_0} \\
\nabla \cdot \vec{B} &= 0 \\
\nabla \times \vec{E} &= -\frac{\partial \vec{B}}{\partial t} \\
\nabla \times \vec{B} &= \mu_0\vec{J} + \mu_0\varepsilon_0\frac{\partial \vec{E}}{\partial t}
\end{aligned}
`} />
