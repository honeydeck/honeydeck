# Why Honeydeck?

<RevealGroup>

- **MDX-native** — Markdown + React, tech you know
- **Static export** — Host anywhere, no runtime JS needed
- **Open format** — Plain text forever, export to PDF anytime
- **AI-ready** — Agent skills for writing and migration

# Theme Tokens

Override CSS variables to brand any deck. The Bee theme brings deep purple and golden yellow. Build your own with `oklch` relative color syntax.

```css
--honeydeck-primary: #6d4aff;
--honeydeck-accent: #ffd93d;
--honeydeck-background: #11101a;
```
